# Prioritized Roadmap

## MVP (v1.0) — 6 weeks
### Week 1-2: Onboarding and DX (P0)
- One-command bootstrap.
- Preflight checks.
- Smoke test automation.
- Preset profiles for first-time users.

### Week 3-4: Commercial Surface (P0/P1)
- Packaging boundaries (CE/Pro/Enterprise).
- License validation hooks.
- Website + docs portal structure.

### Week 5: Deployment Hardening (P0)
- VPS + Coolify guides tested end-to-end.
- Backup/restore validation.
- Security defaults checklist.

### Week 6: Launch Prep (P1)
- Screenshots, architecture diagrams, case studies.
- Pricing page and onboarding docs.
- v1 launch checklist and release candidate freeze.

## Post-v1 Backlog
- Hosted control plane dashboards.
- Upgrade assistant and drift detection.
- Kubernetes reference path (Helm + operators later).
- Enterprise SSO and audit export enhancements.
