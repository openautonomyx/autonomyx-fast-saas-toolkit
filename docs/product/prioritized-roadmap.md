# Prioritized Roadmap (v1 Launch)

## MVP Scope (must-have before launch)

1. Unified onboarding (`init` + `onboard` + dashboard wizard).
2. Environment schema validation + preflight checks.
3. Reliable deployment pathways: local, VPS, Coolify.
4. Docs portal with role-based quickstarts.
5. CE/Pro/Enterprise packaging + entitlement enforcement hooks.
6. Health dashboard with actionable remediation.

## P0 Critical Fixes (next 2–4 weeks)

- Resolve docs/CLI mismatch around `fast-saas deploy`.
- Implement tenant creation flow in dashboard (remove disabled placeholder).
- Introduce canonical config schema package used by CLI + API + dashboard.
- Add smoke tests for generated starter stacks.

## P1 (post-MVP hardening)

- Hosted control plane initial beta (licenses + support intake + release notices).
- Sample tenant/demo seed improvements for sales demos.
- Case studies and screenshots integrated into website/docs.

## P2 (growth)

- Kubernetes reference path (Helm chart alpha).
- In-app upgrade nudges and conversion analytics.
- Enterprise workflow automation (SLA routing, audit exports).

## 90-Day Milestones

### Milestone 1 (Days 1–30)
- Config schema unification.
- Preflight + deploy command implementation.
- Onboarding UX alpha.

### Milestone 2 (Days 31–60)
- Docs portal launch.
- Pricing packaging and entitlement beta.
- Health dashboard remediation playbooks.

### Milestone 3 (Days 61–90)
- Hosted control plane beta.
- Launch assets (videos, case studies, matrix, checklist).
- v1.0 release candidate + launch dry run.
