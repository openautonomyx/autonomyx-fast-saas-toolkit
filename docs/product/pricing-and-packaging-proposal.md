# Pricing and Packaging Proposal

## Edition Strategy

## 1) Community Edition (Free)
**Who:** developers evaluating or bootstrapping.
- Core self-hosted stack.
- Basic CLI and module profiles.
- Community docs + GitHub support.
- No premium modules, no SLA.

## 2) Pro ($99-$499/mo, usage-tiered)
**Who:** startups and agencies shipping production workloads.
- Everything in Community.
- Premium automation packs and onboarding templates.
- License-gated advanced modules/features.
- Priority email support.
- Upgrade channel + migration tooling.

## 3) Enterprise (Custom annual)
**Who:** larger orgs with compliance/support requirements.
- Everything in Pro.
- SSO/SAML controls, enterprise policy packs.
- Dedicated onboarding + architecture review.
- SLA-backed support and incident workflow.
- Private modules and deployment advisory.

## Monetization Hooks
- Runtime license key checked by control-plane entitlement API.
- Feature flags map module/feature to edition.
- Usage-based support add-ons (implementation hours, premium response time).
